Sobre

Esta é a página sobre a aplicação TaskFlow.

O projeto TaskFlow é uma aplicação web desenvolvida com tecnologias Jakarta EE, destinada a gerenciar tarefas de forma eficiente.

Funcionalidades

  • Cadastro de tarefas com título, descrição e data de vencimento.
  • Listagem de tarefas pendentes e concluídas.
  • Marcação de tarefas como concluídas.
  • Exclusão de tarefas.

Tecnologias Utilizadas

  • Jakarta EE 11
  • Servlets
  • JSP (JavaServer Pages)
  • HTML, CSS e JavaScript

Objetivo do Projeto

O objetivo do projeto TaskFlow é fornecer uma ferramenta simples e intuitiva para o gerenciamento de tarefas, permitindo que os usuários organizem suas atividades diárias de maneira eficiente. A aplicação busca facilitar o acompanhamento de tarefas, promovendo maior produtividade e organização pessoal.
